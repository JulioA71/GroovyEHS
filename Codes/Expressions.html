GROOVY EXPRESSIONS

Agregate Expressions

Average

Return the average of the values entered in the values argument.

Expression 


SGS.average(values);

Arguments

  • (Numeric Array) values: Numeric array of values

Example 


return SGS.average(${7439-97-6;Standard});

Min

Return the minimum (smallest) value from the values entered in the values argument.

Expression 


SGS.min(values);

Arguments

  • (Numeric Array) values: Numeric array of values

Example 


return SGS.min(${SampledVolume;Standard});

Max

Return the maximum (largest) value from the values entered in the values argument.

Expression 


SGS.max(values);

Arguments

  • (Numeric Array) values: Numeric array of values

Example 


return SGS.max(${SampledVolume;Standard});

Sum

Return the sum of the values entered in the values argument.

Expression 


SGS.sum(values);

Arguments

  • (Numeric Array) values: Numeric array of values

Example 


return SGS.sum(${7439-97.6;Standard});

Concatenation

Return a string with the values entered in the values argument, separated by the value entered in the separator argument.

Expression 


SGS.concat(values, separator)

Arguments

  • (String Array) values: String array of values
  • (String) separator: String that will be used as separator (use empty string '' if none is desired)

Example 


return SGS.concat(${Appearance;Standard}, ';');

Distinct concatenation

Return a string with the distinct (ignore duplicates) values entered in the values argument, separated by the value entered in the separator argument.

Expression 


SGS.distinctConcat(values, separator);

Arguments

  • (String Array) values: String array of values
  • (String) separator: String that will be used as separator (use empty string '' if none is desired)

Example 


return SGS.distinctConcat(${Appearance;Standard}, ';');

Internal Sample Expressions

Note: Datasets with status "Cancelled" and results marked as Outlier are excluded from the results.

Get SDIDataItem

Return a DataItemInfo object. Used as an argument on several expressions.

Expression 


SGS.getSDIDataitem(datasourceid, sampleid, paramlistid, paramlistversionid, variantid, dataset, paramid, paramtype, replicateid);

Arguments

  • (String) datasourceid: 'labvantage' (EHS) or 'pbluelv8' (LVP)
  • (String) sampleid: Sample ID or ${primary:s_sampleid}
  • (String) paramlistid: Parameter List ID, ${paramlistid}, or '*' (all)
  • (String) paramlistversionid: Parameter List version, ${paramlistversionid}.toString(), or '*' (all)
  • (String) variantid: Parameter List variant, ${variantid}, or '*' (all)
  • (String) dataset: Dataset number, ${dataset}.toString(), '*' (all), or 'max'
  • (String) paramid: Parameter ID, or '*' (all)
  • (String) paramtype: Parameter type, or '*' (all)
  • (String) replicateid: Replicate number, '*', or 'max'

Example 


// Create DataItemInfo object
def SDIDataItem = SGS.getSDIDataitem('labvantage', ${primary:s_sampleid}, ${paramlistid}, ${paramlistversionid}.toString(), ${variantid}, ${dataset}, 'Appearance', 'Standard', 'max');
// Return a DataItemInfo object's property
return SDIDataItem.paramid;

Check if results exist

Return a boolean (true or false). Check if there's a valid result in the specified parameter.

Expression 


SGS.checkResults(datasourceid, sdidataitem, paramlistid, paramlistversionid, variantid, dataset, paramid, paramtype, replicateid, sameWorkitem);

Arguments

  • (String) datasourceid: 'labvantage' (EHS) or 'pbluelv8' (LVP)
  • (DataItemInfo) sdidataitem: DataItemInfo object obtained from SGS.getSDIDataitem expression
  • (String) paramlistid: Parameter List ID, ${paramlistid}, '*' (all), or '#' (same)
  • (String) paramlistversionid: Parameter List version, ${paramlistversionid}.toString(), '*' (all), or '#' (same)
  • (String) variantid: Parameter List variant, ${variantid}, '*' (all), or '#' (same)
  • (String) dataset: Dataset number, ${dataset}.toString(), '*' (all), '#' (same), or 'max'
  • (String) paramid: Parameter ID, '*' (all), or '#' (same)
  • (String) paramtype: Parameter type, '*' (all), or '#' (same)
  • (String) replicateid: Replicate number, '*' (all), '#' (same), or 'max'
  • (Boolean) sameWorkitem: true if the Parameter List/Parameter is in the same Work Item (Test Method), false if it's not in the same WI/TM or if the PL was manually added

Example 


// Create DataItemInfo object
def SDIDataItem = SGS.getSDIDataitem('labvantage', ${primary:s_sampleid}, ${paramlistid}, ${paramlistversionid}.toString(), ${variantid}, ${dataset}, 'Appearance', 'Standard', 'max');
// Return the boolean if the result exists or not
return SGS.checkResults('labvantage', SDIDataItem, 'Analysis', '*', '#', 'max', '#', 'Standard', 'max', true);

Get numeric result from a sample

Return a array of numeric values (either from the entered value or the transformed value, if the Transformation Rule is applied) contained in the specified parameter from one or more sample ID. Allow use of array properties.

Expression 


SGS.sampleNumericResults(datasourceid, sdidataitem, sampleid, paramlistid, paramlistversionid, variantid, dataset, paramid, paramtype, replicateid);

Arguments

  • (String) datasourceid: 'labvantage' (EHS) or 'pbluelv8' (LVP)
  • (DataItemInfo) sdidataitem: DataItemInfo object obtained from SGS.getSDIDataitem expression
  • (String) sampleid: Sample IDs (separated by ;) or ${primary:s_sampleid}
  • (String) paramlistid: Parameter List ID, ${paramlistid}, '*' (all), or '#' (same)
  • (String) paramlistversionid: Parameter List version, ${paramlistversionid}.toString(), '*' (all), or '#' (same)
  • (String) variantid: Parameter List variant, ${variantid}, '*' (all), or '#' (same)
  • (String) dataset: Dataset number, ${dataset}.toString(), '*' (all), '#' (same), or 'max'
  • (String) paramid: Parameter ID, '*' (all), or '#' (same)
  • (String) paramtype: Parameter type, '*' (all), or '#' (same)
  • (String) replicateid: Replicate number, '*' (all), '#' (same), or 'max'

Example 


// Create DataItemInfo object
def SDIDataItem = SGS.getSDIDataitem('labvantage', ${primary:s_sampleid}, ${paramlistid}, ${paramlistversionid}.toString(), ${variantid}, ${dataset}, 'Appearance', 'Standard', 'max');
// Get the child samples and concatenate using ; as a separator
def childSamples = SGS.concat(SGS.childSampleId('labvantage', ${primary:s_sampleid}, ""), ';');
// Return the first position of the array
return SGS.sampleNumericResults('labvantage', SDIDataItem, childSamples, 'Analysis', '*', '#', 'max', '#', 'Standard', 'max', true)[0];

Get Attribute object

Return an Attribute object array. Values can be accessed using the object properties for each array position.

Expression 


SGS.getAttributResults(datasourceid, sdidataitem, attributeid, paramlistid, paramlistversionid, variantid, dataset, paramid, paramtype, replicateid, sameWorkitem);

Arguments

  • (String) datasourceid: 'labvantage' (EHS) or 'pbluelv8' (LVP)
  • (DataItemInfo) sdidataitem: DataItemInfo object obtained from SGS.getSDIDataitem expression
  • (String) attributeid: Attribute ID, or '*'
  • (String) paramlistid: Parameter List ID, ${paramlistid}, '*' (all), or '#' (same)
  • (String) paramlistversionid: Parameter List version, ${paramlistversionid}.toString(), '*' (all), or '#' (same)
  • (String) variantid: Parameter List variant, ${variantid}, '*' (all), or '#' (same)
  • (String) dataset: Dataset number, ${dataset}.toString(), '*' (all), '#' (same), or 'max'
  • (String) paramid: Parameter ID, '*' (all), or '#' (same)
  • (String) paramtype: Parameter type, '*' (all), or '#' (same)
  • (String) replicateid: Replicate number, '*' (all), '#' (same), or 'max'
  • (Boolean) sameWorkitem: true if the Parameter List/Parameter is in the same Work Item (Test Method), false if it's not in the same WI/TM or if the PL was manually added

// Create DataItemInfo object
def SDIDataItem = SGS.getSDIDataitem('labvantage', ${primary:s_sampleid}, ${paramlistid}, ${paramlistversionid}.toString(), ${variantid}, ${dataset}, 'Appearance', 'Standard', 'max');
// Return the numeric value for the Attribute 'Signal'
return SGS.getAttributResults('labvantage', SDIDataItem, 'Signal', 'Analysis', '*', '#', 'max', '#', 'Standard', 'max', true)[0].numericvalue;

External Sample Expressions

Note: Samples with status "Cancelled" and hidden sampels are excluded from the results.

Get child sample ID

Return an array with the child samples of the specified parent sample. Results can be filtered with SQL.

Expression 


SGS.childSampleId(datasourceid, sampleid, condition);

Arguments

  • (String) datasourceid: 'labvantage' (EHS) or 'pbluelv8' (LVP)
  • (String) sampleid: Sample ID or ${primary:s_sampleid}
  • (String) condition: SQL query specifying the conditions to filter the child samples.
    • Enter text between double quotes (")
    • If no condition is needed, enter empty quotes ("")
    • SQL alias:
      • Sample table: s
      • Child sample table: sm
    • Note: The expression SGS.childPlanSampleId uses the same arguments and returns the same data as SGS.childSampleId, but exposes the following tables/alias:
      • Child sample plan table: csp
      • Child sample plan item table: cspi

Example 


// Return the sample ID for the child sample with sample.samplesubtypeid = 'Air_Front'
return SGS.childSampleId('labvantage', ${primary:s_sampleid}, "s.samplesubtypeid = 'Air_Front")[0];

Get QC sample ID

Return an array with the QC sample ID (unknown excluded) with the specified conditions. Results can be filtered with SQL.

Expression 


SGS.qcSampleId(datasourceid, sdidataitem, linked, batchitemtype, condition);

Arguments

  • (String) datasourceid: 'labvantage' (EHS) or 'pbluelv8' (LVP)
  • (DataItemInfo) sdidataitem: DataItemInfo object obtained from SGS.getSDIDataitem expression
  • (Boolean) linked: true return only QCs with linked sample results (i.e., MS, MSD, DUP etc.)
  • (String) batchitemtype: QC sample type, separated by ;, or * (all)
  • (String) condition: SQL query specifying the conditions to filter the QC samples.
    • Enter text between double quotes (")
    • if no condition is needed, enter empty quotes ("")
    • SQL alias:
      • Sample table: s
      • QC batch table: qcb
      • QC batch item table: qcbi
      • QC batch sample type table: qcbst

Example 


// Create DataItemInfo object
def SDIDataItem = SGS.getSDIDataitem('labvantage', ${primary:s_sampleid}, ${paramlistid}, ${paramlistversionid}.toString(), ${variantid}, ${dataset}, 'Appearance', 'Standard', 'max');
// Return the QC samples of type MS and MSD
return SGS.qcSampleId('labvantage', SDIDataItem, true, 'MS;MSD', "")[0];

    • if no condition is needed, enter empty quotes ("")
    • SQL alias:
      • Sample table: s
      • QC batch table: qcb
      • QC batch item table: qcbi
      • QC batch sample type table: qcbst

Example 


// Return the QC samples of type MS and MSD for the current instance of the Work Item EPA8260_W
return SGS.qcSampleIdByWorkItem('labvantage', ${primary:s_sampleid}, 'EPA8260_W', ${#;#;#;#|sourceworkiteminstance}, true, 'MS;MSD', "")[0];
