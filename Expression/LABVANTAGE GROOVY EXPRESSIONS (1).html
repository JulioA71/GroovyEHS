1. Agregate Functions

↑ Return to ToC

Aggregate functions are used to perform a calculation on a set of values (multiple rows) and return a single, summary value.

1.1. Average

1.1.1. Average (SGS.average)

Return the average of the values entered in the values argument.

Expression:

SGS.average(values);

Arguments:

  • (Numeric Array) values: Numeric array of values.

Example:

/*
* values: 10 12 10 10 13
* output: 11
*/
return SGS.average(${7439-97-6;Standard;*});

1.1.2. Average (SGS.averageGtEqTo)

It calculates the average of values (values) that are greater than or equal to a specified limit (limit), optionally replacing values below the limit with zero (replaceByZero).

Expression:

SGS.averageGtEqTo(values, limit, replaceByZero);

Arguments:

  • (Numeric Array) values: Numeric array of values.
  • (Numeric) limit:
  • (Boolean) replaceByZero: true or false if values below the limit are replaced by 0 (zero) or disregarded

Example:

/*
* values: 10.0, 12.0, 10.0, 10.0, 13.0
* limit: 11
* replaceByZero: true
* ouput: 5
*/
SGS.averageGtEqTo(${7439-97-6;Standard;*}, 11, true);

1.2. Minimum (SGS.min)

Return the minimum (smallest) value from the values entered in the values argument.

Expression:

SGS.min(values);

Arguments:

  • (Numeric Array) values: Numeric array of values.

Example:

/*
* values: 10 12 10 9 13
* output: 9
*/
return SGS.min(${SampledVolume;Standard;*});

1.3. Maximum (SGS.max)

Return the maximum (largest) value from the values entered in the values argument.

Expression:

SGS.max(values);

Arguments:

  • (Numeric Array) values: Numeric array of values.

Example:

/*
* values: 10 12 10 9 13
* output: 13
*/
return SGS.max(${SampledVolume;Standard;*});

1.4. Sum (SGS.sum)

Return the sum of the values entered in the values argument.

Expression:

SGS.sum(values);

Arguments:

  • (Numeric Array) values: Numeric array of values.

Example:

/*
* values: 10 12 10 9 13
* output: 54
*/
return SGS.sum(${7439-97-6;Standard;*});

1.5. Concatenation

1.5.1 Simple concatenation (SGS.concat)

Return a string with the values entered in the values argument, separated by the value entered in the separator argument.

Expression:

SGS.concat(values, separator)

Arguments:

  • (String Array) values: String array of values.
  • (String) separator: String that will be used as separator (use empty string '' if none is desired).

Example:

/*
* values: apple pear grape pear apple
* separator: ;
* output: apple;pear;grape;pear;apple
*/
return SGS.concat(${Appearance;Standard;*}, ';');

1.5.2. Distinct concatenation (SGS.distinctConcat)

Return a string with the distinct (ignore duplicates) values entered in the values argument, separated by the value entered in the separator argument.

Expression:

SGS.distinctConcat(values, separator);

Arguments:

  • (String Array) values: String array of values.
  • (String) separator: String that will be used as separator (use empty string '' if none is desired).

Example:

/*
* values: apple pear grape pear apple
* separator: ;
* output: apple;pear;grape
*/
return SGS.distinctConcat(${Appearance;Standard;*}, ';');

1.6. Transform to array (SGS.toArray)

Transform a list of numeric list of values to an array.

Expression:

SGS.toArray(list)

Arguments:

  • (Numeric Array) values: Numeric array of values.
  • (String) separator: String that will be used as separator (use empty string '' if none is desired).

Example:

/*
* list: 10 9 10 11 12
* output: [10, 9, 10, 11, 12]
*/
return SGS.toArray(${7439-97-6;Standard;*});

2. Internal Sample Functions

↑ Return to ToC

Internal sample functions are used to return values belonging or associated to the current or specified sample.
Datasets with status "Cancelled" and parameters marked as Outlier are excluded from the results.

2.1. Get DataItemInfo object (SGS.getSDIDataitem)

Return a DataItemInfo object that can be used as an argument on several functions.

Expression:

SGS.getSDIDataitem(datasourceid, sampleid, paramlistid, paramlistversionid, variantid, dataset, paramid, paramtype, replicateid);

Arguments:

  • (String) datasourceid: 'labvantage' (EHS) or 'pbluelv8' (LVP).
  • (String) sampleid: Sample ID or ${primary:s_sampleid}.
  • (String) paramlistid: Parameter List ID, or ${paramlistid}.
  • (String) paramlistversionid: Parameter List version, ${paramlistversionid}.toString(), or '*' (all).
  • (String) variantid: Parameter List variant, ${variantid}, or '*' (all).
  • (String) dataset: Dataset number, ${dataset}.toString(), '*' (all), or 'max'.
  • (String) paramid: Parameter ID, or '*' (all).
  • (String) paramtype: Parameter type, or '*' (all).
  • (String) replicateid: Replicate number, '*', or 'max'.

Example:

// Create DataItemInfo object
def SDIDataItem = SGS.getSDIDataitem('labvantage', ${primary:s_sampleid}, ${paramlistid}, ${paramlistversionid}.toString(), ${variantid}, ${dataset}, 'Appearance', 'Standard', 'max');
// Return a DataItemInfo object's property
return SDIDataItem.paramid;

Object properties:

Property Data Type Description
sdcid String SDC ID
keyid1 String Sample ID
keyid2 String
keyid3 String
paramlistid String Parameter List
paramlistversionid String Parameter List version
variantid String Parameter List variant
dataset Integer Dataset number
paramid String Parameter ID
paramtype String Parameter type
replicateid Integer Replicate number
datatypes String Data type
aliasid String Parameter alias
releasedflag String Parameter Released flag
reportflag String Parameter set to reportable flag
activeflag String
calcexcludedflag String Parameter marked as outlier
sourceworkitemid String Test Method ID
sourceworkiteminstance String Test Method instance
s_qcbatchid String
s_qcbatchitemid String

2.2. Check if results exist (SGS.checkResults)

Check if there's a valid result in the specified parameter and return true or false.

Expression:

SGS.checkResults(datasourceid, sdidataitem, paramlistid, paramlistversionid, variantid, dataset, paramid, paramtype, replicateid, sameWorkitem);

Arguments:

  • (String) datasourceid: 'labvantage' (EHS) or 'pbluelv8' (LVP).
  • (DataItemInfo) sdidataitem: DataItemInfo object obtained from SGS.getSDIDataitem function.
  • (String) paramlistid: Parameter List ID, ${paramlistid}, '*' (all), or '#' (same).
  • (String) paramlistversionid: Parameter List version, ${paramlistversionid}.toString(), '*' (all), or '#' (same).
  • (String) variantid: Parameter List variant, ${variantid}, '*' (all), or '#' (same).
  • (String) dataset: Dataset number, ${dataset}.toString(), '*' (all), '#' (same), or 'max'.
  • (String) paramid: Parameter ID, '*' (all), or '#' (same).
  • (String) paramtype: Parameter type, '*' (all), or '#' (same).
  • (String) replicateid: Replicate number, '*' (all), '#' (same), or 'max'.
  • (Boolean) sameWorkitem: true or false if the Parameter List/Parameter is in the same Work Item (Test Method) from the DataItemInfo object.

Example:

// Create DataItemInfo object
def SDIDataItem = SGS.getSDIDataitem('labvantage', ${primary:s_sampleid}, ${paramlistid}, ${paramlistversionid}.toString(), ${variantid}, ${dataset}, 'Appearance', 'Standard', 'max');
// Return the boolean if the result exists or not
return SGS.checkResults('labvantage', SDIDataItem, 'Analysis', '*', '#', 'max', '#', 'Standard', 'max', true);

  • (String) paramid: Parameter ID, '*' (all), or '#' (same).
  • (String) paramtype: Parameter type, '*' (all), or '#' (same).
  • (String) replicateid: Replicate number, '*' (all), '#' (same), or 'max'.
  • (Boolean) sameWorkitem: true or false if the Parameter List/Parameter is in the same Work Item (Test Method) from the DataItemInfo object.

External sample functions are used to return values related to child samples and QC samples.

3.1. Child sample

3.1.1. Get child sample ID (SGS.childSampleId)

Return an array with the child sample IDs of the specified parent sample. Results can be filtered with SQL.

Expression:

// Child sample ID
SGS.childSampleId(datasourceid, sampleid, condition);

Arguments:

  • (String) datasourceid: 'labvantage' (EHS) or 'pbluelv8' (LVP).
  • (String) sampleid: Sample ID or ${primary:s_sampleid}.
  • (String) condition: SQL query specifying the conditions to filter the child samples.
    • Enter text between double quotes (").
    • If no condition is needed, enter empty quotes ("").

// Return the sample ID for the child sample with sample.samplesubtypeid = 'Air_Front'
return SGS.childSampleId('labvantage', ${primary:s_sampleid}, "s.samplesubtypeid = 'Air_Front")[0];

4.1. Statistics

4.1.1. Inverse of the chi-squared distribution (SGS.ChiInv)

Returns the inverse of the right-tailed probability of the chi-squared distribution.

Expression:

SGS.ChiInv(probability, deg_freedom);

Arguments:

  • (Number) probability: Probability.
  • (Number) deg_freedom: Degree of freedom.

Example:

// Return ChiInv for the probability and degree of freedom
return SGS.ChiInv(${Probability;Final Value}, ${DegOfFreedom;Final Value});

4.1.2. Poisson Distribution (SGS.PoissonDist)

Returns the Poisson distribution.

Expression:

SGS.PoissonDist(x, mean, cumulative);

Arguments:

  • (Number) x: Number of events.
  • (Number) mean: Expected numeric value.
  • (Boolean) cumulative: A logical value that determines the form of the probability distribution returned:
    • true: returns the cumulative Poisson probability that the number of random events occurring will be between zero and x inclusive.
    • false returns the Poisson probability mass function that the number of events occurring will be exactly x.

Example:

// Return poisson distribution
return SGS.PoissonDist(${Events;Final Value}, ${Mean;Final Value}, true);

Arguments:

  • (Number) b: Second value.

Example:
